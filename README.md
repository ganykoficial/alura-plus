# Alura Plus - Página Promocional

## Descrição

Este é um projeto de uma página promocional para o serviço Alura Plus, que combina o acesso aos cursos da Alura+ e Alura Língua. A página apresenta informações sobre o combo promocional, dispositivos compatíveis, e os principais recursos oferecidos pela plataforma. O layout é responsivo e utiliza variáveis CSS para facilitar a manutenção de cores e estilos.

## Tecnologias

- **HTML5**: Estrutura semântica do documento.
- **CSS3**: Estilos modernos com uso de variáveis para cores e fontes.
- **Google Fonts**: Fontes otimizadas com a família de fontes "Inter".

## Funcionalidades

- **Botões Interativos**: A página inclui botões de ação com transições suaves e mudanças de cor ao passar o mouse.
- **Design Responsivo**: A página foi otimizada para funcionar em diferentes tamanhos de tela (TV, computadores, celulares).
- **Uso de Variáveis CSS**: Facilita a mudança de cores em toda a página sem a necessidade de alterar múltiplos pontos no código.

## Estrutura do Projeto

- **index.html**: Estrutura principal da página.
- **style.css**: Estilos personalizados com variáveis CSS.
- **IMG/**: Pasta contendo as imagens utilizadas na página (ex: Combo, Plataformas, Telas, etc.).
